# ECharts Lab 快速启动指南

## 🚀 一键启动

### Windows 用户

双击运行以下文件即可：

```
start.bat
```

服务会在后台启动，浏览器会自动打开。

### 停止服务

双击运行：

```
stop.bat
```

---

## 📝 访问地址

- **前端界面**: http://localhost:5173
- **后端API**: http://localhost:8001
- **默认账号**: admin / admin123

---

## 🔧 手动启动 (PowerShell)

如果 .bat 文件无法运行，可以使用 PowerShell：

```powershell
.\start.ps1
```

停止服务：

```powershell
.\stop.ps1
```

---

## ⚠️ 常见问题

### 1. 提示"禁止运行脚本"

**解决方案**: 使用 .bat 文件代替 .ps1 文件

### 2. 端口被占用

**解决方案**: 
- 先运行 `stop.bat` 停止现有服务
- 或手动关闭占用 5173/8001 端口的程序

### 3. 虚拟环境未找到

**解决方案**:
```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
.venv\Scripts\activate

# 安装依赖
cd backend
pip install -r requirements.txt

cd ..\frontend
npm install
```

---

## 📦 服务说明

启动脚本会：
1. ✓ 检查虚拟环境是否存在
2. ✓ 检查端口是否被占用
3. ✓ 在后台启动前端服务 (Vite)
4. ✓ 在后台启动后端服务 (FastAPI)
5. ✓ 自动打开浏览器

所有进程都在后台运行，不会显示黑窗口。
